# 🐄 Projeto Agrinho 2026 - Leite Sustentável

Este projeto foi desenvolvido para o **Concurso Agrinho 2026**, sob o tema *"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"*. O objetivo é conscientizar e educar sobre as boas práticas no manejo do gado de leite de forma interativa e tecnológica.

O site foi construído utilizando o conceito de **SPA (Single Page Application)**, onde todo o conteúdo é carregado em uma única página de forma rápida, utilizando apenas HTML, CSS e JavaScript puros.

## 🕹️ Como Usar o Web Site (Manual de Orientações)

O site é dividido em 3 seções principais que podem ser acessadas pelo menu de navegação no topo:

### 1. Simulador & Manejo (Página Inicial)
* **O Simulador de Ordenha:** No centro da tela, você verá um botão vermelho escrito **"Tirar Leite"** e um balde cinza. 
  * Cada vez que você clica no botão, o leite sobe 10% no balde (simulando a ordenha).
  * Quando o balde atinge 100%, uma mensagem de sucesso aparece: *"Balde cheio! Excelente produção. 🥛"*.
  * Se clicar novamente com o balde cheio, ele se esvazia para que você possa começar uma nova simulação.
* **Guia do Produtor Sustentável:** Logo abaixo do simulador, há cartões informativos detalhando:
  * O que aumenta e o que diminui a produção de leite (focando no bem-estar animal).
  * Como deve ser o manejo rotacionado do pasto para proteger o meio ambiente.
  * Cuidados com a saúde do gado (vacinas e controle de pragas).
  * A idade correta e manejos para apartar os bezerros de forma humanizada.

### 2. Sobre Mim
* Esta seção apresenta o autor do projeto (aluno do 3º ano do Ensino Médio) e as motivações por trás da escolha do tema e o uso da tecnologia no campo.

## 🌱 Alinhamento com o Tema do Agrinho 2026

O simulador e os textos educativos reforçam que **o agro forte depende da sustentabilidade**. O bem-estar da vaca (sombra, água limpa, sem estresse) e o cuidado com o solo (pasto rotacionado) são os pilares que provam que é possível ter uma alta produção de leite mantendo o equilíbrio perfeito com o meio ambiente.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização responsiva, cores temáticas (tons de verde e terra) e animação do balde de leite.
* **JavaScript (Vanilla):** Lógica de navegação entre seções (SPA) e o funcionamento interativo do simulador de ordenha.
